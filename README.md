# MAPPA: a video mapping toolset
This library was developed over the course of two years as a general purpose mapping tool.
It makes use of different mapping libs and other resources, and acts as a wrapper to them.
While also implementing easy to use features in order to make projection mapping for experimental arts interesting!

##Used Libraries
- Keystone by David Bouchard
- KinectPV2 by Thomas Sanchez Lengeling
- OscP5 and NetP5 by Andreas Schlegel
- Spout by Lynn Jarvis and Martin Froehlich
- Video by The Processing Foundation

Without the longlasting support for these libraries, this would not be possible.
